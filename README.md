### Tech stack:
___

- Vite 4.4.9
- Node.js >14.0.0
- Handlebars (vite-plugin-handlebars)
- Typescript 5.2.2
- Postcss(plugin postcss-preset-env)

### How to use
___

 use ```npm run dev``` - for start development <br>
 use ```npm run start``` - for build and start server
